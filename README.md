### Step 4: 
End the program
## Program:

#Program to find the rank of a matrix.

#Developed by: S.Dhayalaprabu

#RegisterNumber:24006289
```
import numpy as np
A=np.array([[3,2,5],[1,1,2],[3,3,6]])
result=np.linalg.matrix_rank(A)
print(result)
```
